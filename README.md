# KNN Iris Classification

## 📌 Project Overview
This project demonstrates the implementation of the K-Nearest Neighbors (KNN) algorithm using the Iris dataset. It includes feature normalization, experimenting with different values of K, evaluating model performance, and visualizing decision boundaries.

## 🎯 Objectives
- Apply KNN for classification
- Normalize features for better performance
- Experiment with different K values
- Evaluate model using accuracy and confusion matrix
- Visualize decision boundaries

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📂 Dataset
- Iris Dataset (CSV file)

## ⚙️ Steps Performed
1. Loaded the dataset using Pandas  
2. Split features and target  
3. Applied feature normalization using StandardScaler  
4. Split data into training and testing sets  
5. Trained KNN model with different values of K  
6. Evaluated model using:
   - Accuracy Score
   - Confusion Matrix  
7. Visualized decision boundaries using 2 features  

## 📊 Results
- Achieved high accuracy on the Iris dataset  
- Observed how different K values affect performance  
- Decision boundaries clearly show class separation  

## 📷 Visualization
Decision boundary plots help in understanding how KNN classifies different regions in feature space.
